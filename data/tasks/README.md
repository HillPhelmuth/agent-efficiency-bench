# Dataset notes

`public_efficiency_subset.jsonl` is generated by:

```bash
PYTHONPATH= uv run aeb build-subset --config configs/sources.yaml --output data/tasks/public_efficiency_subset.jsonl
```

The file is intentionally small and stores normalized task metadata/instructions only. It does **not** vendor full benchmark sandboxes, archives, repositories, or hidden test answers.

Current generated subset:

- 8 tasks from `SWE-bench/SWE-bench_Lite` via Hugging Face.
- 8 tasks from `AssistantBench/AssistantBench` via Hugging Face raw JSONL.
- 8 tasks from `harbor-framework/terminal-bench` via GitHub task YAML files.
